# Welcome to My Blog!

I'm excited to launch this blog as part of my portfolio site. This is where I'll be sharing my thoughts on:

- **Software Engineering** - Best practices, interesting problems, and lessons learned
- **Teaching & Education** - Reflections from my time as a Teaching Fellow at Harvard
- **Data Science** - Projects and insights from working with data
- **Personal Projects** - Updates on side projects and experiments

## What to Expect

I'll aim to post regularly about topics that interest me and hopefully provide value to others in the tech community. Whether you're a student, fellow developer, or just curious about tech, I hope you'll find something useful here.

## How This Blog Works

This blog is built with Jekyll, a static site generator that lets me write posts in Markdown. Each post is simply a Markdown file that gets automatically converted to HTML and deployed via GitHub Pages.

The workflow is simple:
1. Create a new `.md` file in `_posts/`
2. Write the post in Markdown
3. Commit and push to GitHub
4. GitHub Pages automatically builds and deploys the site
